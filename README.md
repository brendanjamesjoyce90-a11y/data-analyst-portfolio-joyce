# My Research Topics
My portfolio examines two questions about international security and military technology diffusion.
- *How did the international community respond leading up to and after Russia's invasion of Ukraine on February 24, 2022, and which states drove that response?*
- *Second, how quickly have armed UAVs spread compared to other advanced military platforms, and what does that tell us about the changing nature of war?*

# The Data Sources
- For Problem Set 2, I used the *International Response to the Ukraine War* dataset. It tracks diplomatic, military, and economic actions by states from January–April 2022. You can find this dataset at Janes Intelligence Review’s customer portal. (https://customerportal.janes.com/)
- For Problem Set 4, I used the Correlates of War (COW) *Arms Technology Dataset* v1.0 (2025), covering adoption of 52 military technologies across 190 states from 1815–2023. You can find the dataset on COW’s website at https://correlatesofwar.org/

# My Approach to Analysis
- For the revised problem set 2, I filtered the response data by NATO membership and pre/post-invasion timing, then visualized cumulative responses over time and mapped response intensity by country.
- For the revised problem set 4, I extracted UAV adoption years, joined them to world map data, and created a gradient map showing patterns of diffusion. I also compared UAV adoption rates to legacy platforms (5th gen fighters, combat helicopters, tanks) to demonstrate the unprecedented speed of UAV proliferation.
